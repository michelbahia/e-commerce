# Imersão Full Stack & FullCycle - Ecommerce

## Descrição

Repositório do serviço de pagamentos feito com Golang

## Rodar a aplicação

** Requisitos: **

- Docker
- Docker Compose
- Golang


Execute a aplicação

```
go run cmd/payment/main.go
```
